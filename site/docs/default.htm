Ultra

By Steven Goodwin, 2014

Ultra is an ultra small (<64k binary), ultra fast, all-in-memory web server incorporating an in-built NoSQL datastore, a data processing language, SSI, multiple configurations, and logging.

It is intended for IoT and BigData applications.

Basic Functionality

Basic web server, supporting GET requests and argument parsing

NoSQL data store, of name=value pairs, with implicit structure through nesting.the.namespace

In memory storage of all data (site and data store) for ultra fast retrieval

Basic processing of data elements

Server side includes

Alternate configurations for production and staging servers

Use of kill -SIGHUP to reload the data set, and -SIGUSR1 to serialize the database to disc
